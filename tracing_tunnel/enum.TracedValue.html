Enum TracedValue 

Source 
#[non_exhaustive]
pub enum TracedValue {
    Bool(bool),
    Int(i128),
    UInt(u128),
    Float(f64),
    String(String),
    Object(DebugObject),
    Error(TracedError),
}
Expand description

Value recorded in a tracing span or event.
